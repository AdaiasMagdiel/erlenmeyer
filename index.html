    Erlenmeyer Framework¶

    ‚öóÔ∏è A lightweight, modular, and elegant PHP microframework for building modern web applications and APIs.

    üß† What is Erlenmeyer?¶

    Erlenmeyer is a minimal and composable PHP microframework that gives you the essential tools ‚Äî routing, requests, responses, sessions, and testing ‚Äî all without imposing any rigid structure.

    You decide how your project looks.
    Want a clean MVC layout? Perfect.
    Prefer everything in a single index.php file? That works too.

    It draws inspiration from the Flask microframework in Python ‚Äî sharing the same principles of simplicity, explicitness, and developer freedom ‚Äî but redesigned from the ground up for the PHP ecosystem.

    Erlenmeyer is about clarity without constraints.

    !!! quote "The philosophy" > ‚ÄúSimplicity is the ultimate sophistication.‚Äù

    ‚Äî Leonardo da Vinci

    ‚ú® Key Features¶

    Category Highlights
    Routing Simple syntax with dynamic parameters (/users/[id]) and redirects
    Middleware Global and per-route, easy $next($req, $res, $params) chaining
    Requests & Responses Typed API with helpers for HTML, JSON, text, and files
    Logging Console or rotating file-based logs with structured levels
    Error Handling Custom exception mapping and fallback handlers
    Assets Static file delivery with MIME detection and caching
    Sessions Flash messages and persistent data management
    Testing ErlenClient to simulate full HTTP requests for testing

    ‚öôÔ∏è A Quick Example¶

    <?php

use AdaiasMagdiel\Erlenmeyer\Request;
use AdaiasMagdiel\Erlenmeyer\Response;

require __DIR__ . '/bootstrap.php';

$app = new App();

$app->get('/', fn(Request $req, Response $res) =>
    $res->withHtml('<h1>Hello from Erlenmeyer ‚öóÔ∏è</h1>')
);

    $res->withText("Hello, {$params->name}!")

    Run it with:

    php -S localhost:8000

    and visit http://localhost:8000

    üîß The .htaccess (recommended)¶

    RewriteEngine On

# General settings
Options -Indexes
Options +FollowSymLinks

# Disable Apache version exposure
Header always unset X-Powered-By

# Allow access to static files
RewriteRule ^(assets|public)/.* - [L]

# Block direct access to PHP files except index.php
RewriteCond %{REQUEST_URI} !/index\.php$ [NC]
RewriteCond %{REQUEST_URI} \.php$ [NC]
RewriteRule ^ - [R=404,L]

# Redirect all other requests to index.php
RewriteRule ^ index.php [L]

    ‚úÖ This version improves security and readability. (Explicit +FollowSymLinks and clear file-access rules.)
